How to Setup kit-kat-touch.exe (part 2)

Download kit-kat-touch.exe: Here (if you haven't already)

Open kit-kat-touch.exe

  • Click the Downloads Tab (second tab in the right taskbar)

  • Download all Four Program .zips (by clicking the links in kit-kat-touch.exe)

  • Create a Folder named ie: "3DS Streaming"

  • Put kit-kat-touch.exe inside /3DS Streaming/

  • Put the Four Program Zip files Inside /3DS Streaming/

  • Unzip InputRedirectionClient-Qt, Joy2Key, & Vitrite  into /3DS Streaming/ so that the ".exes" are all in the Same Directory (delete the zips afterwords)

 

  • Navigate to the NTR_3.4PREVIEW2_STARTER_KIT Folder

  • Cut NTR_VIEWER Folder

  • Paste the NTR_Viewer Folder in /3DS Streaming/
  • Delete the NTR_3.4PREVIEW2_STARTER_KIT Folder
